RF-Study's API

For admin
POST /api/auth/login
POST /api/auth/register
For students
GET(all students) /api/std
GET(student by Student id) /api/std/:id
POST /api/std/
PUT /api/std/:id
DELETE /api/std/:id
POST(student log) /api/std/:id/logs
For devices
GET(all devices) /api/device
GET(device by id) /api/device/:id
POST /api/device/
PUT /api/device/:id
DELETE /api/device/:id
